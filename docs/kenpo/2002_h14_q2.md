# 平成14年度 旧司法試験 論文式試験問題 憲法 第2問

## 問題

> 1 国会で今制定されようとしているA法律は明らかに違憲であるとして、成立前に無効の宣言をするよう求める訴え。

>
> 3 自衛隊は憲法第9条に違反する無効な存在であるとして、国に対して、自己の納税分中自衛隊に支出した額の返還を請求する訴え。
