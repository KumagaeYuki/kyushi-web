# 平成22年度 旧司法試験 論文式試験問題 憲法 第1問

## 問題

> 理容師法は、「理容師の資格を定めるとともに、理容の業務が適正に行われるように規律し、もって公衆衛生の向上に資することを目的」として制定された法律であり（同法第1条）、同法第12条第4号は、理容所（理髪店）の開設者に対し、「都道府県が条例で定める衛生上必要な措置」を講ずべき義務を課し、第14条は、都道府県知事は、開設者が第12条の規定に違反したときは、期間を定めて、当該理容所の閉鎖を命ずることができる旨を定めている。
>
> A県においては、最近、主要な駅の周辺等に、洗髪設備を持たない簡易な施設を設け、安価かつ迅速に散髪を行うことを売り物にした理容所が数多く開設され、利用者を増やしている反面、従来型の理容所の利用者が減少している。そこで、A県は、理容師法の目的を達成し、理容の業務を適正化するとともに、一層の衛生確保により、公衆衛生の向上を図る目的で、理容師法第12条第4号に基づき、理容所において洗髪を行うのに必要な、給湯が可能な設備を設けることを義務付ける条例を制定した。
>
> このA県条例に含まれる憲法上の問題点について論ぜよ（なお、法律と条例の関係について論ずる必要はない。）。
