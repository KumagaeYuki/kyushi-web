# 平成22年度 旧司法試験 論文式試験問題 憲法 第2問

## 問題

> 選挙区選出の参議院議員甲は、日ごろから本会議で居眠りをしたり、大声で不規則発言を繰り返していたため、議長や所属政党からたびたび注意されていたが、一向に改まらなかった。それどころか、平成22年2月某日、甲は、酒気を帯びて本会議に出席し、不規則発言をするだけでなく、対立政党の議員に対して暴力を振るった。
>
> こうした甲の一連の行為に対し、参議院は、出席議員の3分の2以上の多数による議決により、甲を除名した。甲は、この除名処分を不服として、裁判所に対し、議員の地位の確認を求める訴えを提起した。
>
> 本件に含まれる憲法上の問題点について、甲が地方議会議員であって、同様の理由で地方議会から除名された場合、及び甲が参議院議員であって、所属政党から除名された場合と比較しつつ、論ぜよ。
