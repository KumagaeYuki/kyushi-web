# 平成7年度 旧司法試験 論文式試験問題 刑法 第1問

## 問題

> 甲は、乙から「強盗に使うのでナイフを貸してくれ。」と依頼され、これに応じてナイフを乙に渡した。その後、乙は、丙・丁に対し、「最近、知り合いのAが多額の保険金を手に入れたので、それぞれがナイフを準備してA宅に強盗に押し入ろう。」と持ち掛け、3名で計画を立てた。ところが、乙は、犯行当日の朝になって高熱を発したため、「おれはこの件から手を引く。」と丙・丁に電話で告げて、両名の了承を得た。しかし、丙・丁は予定どおり強盗に押し入り現金を奪った。
>
> 甲及び乙の罪責を論ぜよ（特別法違反の点は除く。）。
