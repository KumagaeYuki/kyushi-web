# 平成22年度 旧司法試験 論文式試験問題 刑法 第2問

## 問題

> 甲は、紳士服専門店A社の営業担当者として高級紳士服の販売を担当していたが、遊興費欲しさから、顧客から金銭を得ようと企てた。
>
> 甲は、来店したBに対し、実際にはスーツを仕立てる意思がないのに、「良いオーダースーツをお作りします。一着20万円です。」と勧誘した。既製品に不満を抱いていたBは、注文することを決め、甲は、生地を選ばせたり、採寸するなどした上、「この生地なら代金は20万円で、内金7万円を預からせてください。仕上がりは今日から4週間後です。」と言った。Bは、これを信じてスーツを注文し、内金として現金7万円を甲に預けたが、甲は、直ちにその全額をパチンコに費消した。
>
> 4週間後、甲は、Bに「スーツが出来上がったので来店してください。」と電話し、A社の店舗に呼び出した。甲は、来店したBを店内で待たせておき、徒歩数分の場所にある倉庫に行き、そこを管理する既製服部門の責任者Cに対し、「チラシ撮影用にスーツを一着借りていく。」と言った。Cは、これを信じて「すぐ返してください。」と言ったため、甲は、Bが選んだ生地に似ていてBの体格に合う既製スーツ（販売価格20万円）を持ち出した。
>
> 甲は、店舗に戻り、「ご注文のスーツです。」と言って、その既製スーツがあたかもオーダースーツであるかのように装ってBに手渡した。Bは、試着してみたところ、ぴったりだったため、既製品であることに気付かず、満足し、残額13万円を現金で支払って持ち帰った。甲は、この13万円も全額自分の飲食代に費消した。
>
> 甲の罪責を論ぜよ。
