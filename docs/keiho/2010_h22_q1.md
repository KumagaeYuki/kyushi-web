# 平成22年度 旧司法試験 論文式試験問題 刑法 第1問

## 問題

> 甲は、かつて働いていたA社に忍び込んで金品を盗もうと考え、親友であるA社の従業員乙にこの計画を打ち明けて、その援助を依頼した。乙は、甲の依頼を快諾し、甲に対し、「社員が退社した後に通用口の鍵を開けておく。」と言い、甲もこれに対して礼を言った。乙は、犯行当日、他の社員が退社した後に、通用口の鍵を開けて帰宅した。
>
> 甲は、バールを持ってA社の前まで来たが、人の気配がしたので、A社に忍び込むのはやめて、隣のB社に忍び込むことに計画を変更した。甲は、たまたま鍵が開いていた玄関ドアから、無人のB社建物内に侵入し、事務室にあった金庫の扉をバールでこじ開けて中の現金を盗み、更に金目の物を探していたところ、誤って書類の束に手を触れ、これを、B社の従業員丙が消し忘れてつけっ放しにしていた石油ストーブの上に落としてしまい、書類の束に火がついた。甲は、近くに消火器があったので、これを使えば容易に消火できると認識したが、消火活動をすれば、盗みが発覚してしまうと思い、そのまま放置して立ち去った。
>
> 一方、丙は、帰宅途中で石油ストーブを消し忘れたことに気付いてB社に戻り、事務室に入ろうとしたところ、床が燃えているのを発見した。その時点では、まだ備え付けの消火器を使えば容易に消すことが可能な状態であったが、丙は、火災が自分の不始末によるものと考え、その責任が発覚するのを恐れて、そのまま立ち去り、その結果、B社の建物は全焼した。
>
> 甲、乙及び丙の罪責を論ぜよ（ただし、特別法違反の点は除く。）。
