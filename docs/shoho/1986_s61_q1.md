# 昭和61年度 旧司法試験 論文式試験問題 商法 第1問

## 問題

> 甲は、乙株式会社（発行済株式総数500万株）の株式一万株を証券会社を通じて購入し、株券を提示して株主名簿の名義書換を請求したが、乙会社は、当該株券につき株主名簿上の株主から盗難届が出されていることを理由に、名義書換を拒絶した。その後、300万株の株式を有する株主が出席した乙会社の株主総会において、200万株の株式を有する株主の賛成で取締役選任の決議がされたが、甲には、その総会の招集通知が発せられていなかった。乙会社の株主丙は、甲に招集通知が発せられなかったことを理由に、その決議の取消しを求める訴えを提起した。
>
> この請求は認められるか。
