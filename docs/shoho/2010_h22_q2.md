# 平成22年度 旧司法試験 論文式試験問題 商法 第2問

## 問題

> A株式会社は、平成16年12月20日、貸金業者であるXから事業資金として3000万円を借り入れた。その際、①A社はXに対し毎月1パーセントの割合による利息を支払うこと、②A社が約定どおり利息の支払をしている間は元本の返済を猶予するが、A社が1回でも利息の支払を怠ったときは、直ちに元本全額を返済すること、③Xの債権回収を確実にするために、A社は、手形金額3000万円、振出日及び満期を白地とする約束手形（以下「本件手形」という。）を振り出すことが合意された。その際、Xから、A社の取締役Yに手形の支払を保証させるよう求められたため、A社の代表取締役Bは、受取人をYとする本件手形を振り出し、Yがこれに拒絶証書作成不要の文言を記載した白地式裏書をして、Xに交付した。
>
> なお、A社においては、事業資金の借入れについては取締役会の決議を経ているが、本件手形の振出しについては取締役会の承認を受けていなかった。
>
> その後、A社は、しばらくの間、約定どおり利息を支払っていたが、平成19年5月ころから資金繰りが悪化し、同年8月以降、利息の支払ができなくなった。そこで、Xは、平成22年4月9日、本件手形の振出日を同月1日、満期を同年5月10日と補充して、満期に本件手形を呈示した上で支払を求めたが、その支払を拒絶されたため、Yに対して手形金の支払を請求した。
>
> Yは、Xの請求を拒むことができるか。
