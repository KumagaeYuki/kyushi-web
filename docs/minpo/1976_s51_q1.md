# 昭和51年度 旧司法試験 論文式試験問題 民法 第1問

## 問題

> 甲は、その所有する建物をAに賃貸して引き渡したが、その際、Aが賃借権を自由に譲渡できる旨の特約をした。甲は、後にその建物を乙に売却し、その旨をAに通知したが、所有権移転登記をしないでいた。乙は、Aに対して家賃を自分に支払うように求めたが、Aはこれを拒んで、家賃を甲のもとに持参した。しかし、甲も受領を拒んだので、Aは、3か月後に賃借権をBに譲渡し、建物を引き渡した。そこで、乙は、直ちにAに対して賃貸借契約の解除の意思表示をし、Bに対して建物の明渡しを求めている。乙・B間で生ずる法律問題を挙げて論ぜよ。
